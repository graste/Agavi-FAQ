SHJS - Download

To use SHJS, you need at least 3 files:

  1. The main script or the minified version
  2. A language definition file
  3. A stylesheet (alternatives)

You can simply download the files you need from the list above, or you can download a binary distribution containing all languages and style sheets. Binary distributions have names of the form shjs-VERSION.zip.

The SHJS binary distribution comes with definitions for over 20 popular programming languages. If you wish to define your own languages, you will need to download a source distribution, which will have a name of the form shjs-VERSION-src.zip. The source distribution contains the sh2js.pl script used for defining new languages. (It also contains all the pre-defined languages and style sheets found in the binary distribution.)
